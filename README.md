This is a simple Django app for demo

REQUIREMENTS
--------------------
* Python 2.x 
* Pip
* SQLite3
* Git


INSTALL
--------------------
    $ git clone https://github.com/sonnguyenthai/GL_Django_Demo.git gldjangodemo
    $ cd gldjangodemo
    $ pip install -r requirements.txt
    $ ./manage.py migrate
    $ ./manage.py runserver

Open your browser and enjoy http://localhost:8000
